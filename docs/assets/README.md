# `docs/assets/` — Media Assets

Screenshots, diagrams, and GIFs for documentation and blog-ready visuals.

## Guidelines

- **Resolution** — 1280 px+ width for readability in docs and slides.
- **Format** — PNG for screenshots, SVG for diagrams, GIF/WebM for short recordings.
- **Naming** — lowercase, hyphen-separated: `trust-graph-overview.png`.

## Suggested Assets

| Filename | Content |
|----------|---------|
| `terminal-chat.png` | Home page CRT terminal with AI explanation panel |
| `tutor-path.png` | Red Team Tutor learning path UI |
| `trust-graph.png` | D3.js trust graph visualization with nodes/edges |
| `walkthrough-demo.gif` | Autonomous walkthrough in action |
| `recon-report.png` | Test & Report findings output |
| `scanner-results.png` | TN3270 network discovery results |
