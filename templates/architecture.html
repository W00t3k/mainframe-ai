System Architecture

A local-first mainframe operations assistant with FastAPI at the center, Ollama for LLM + embeddings, a file-backed RAG store, and a TN3270 pipeline for live terminal control.

Web UI

Jinja templates with shared CSS, toast notifications, and a consistent IBM-style layout.

API Layer

FastAPI endpoints for chat, terminal I/O, RAG ingestion, scanner, recon, and screencaps.

Red Team Tutor

Step-gated learning modules with persona-aware guidance and screen-context analysis.

LLM + RAG

Ollama handles generation and embeddings; the RAG engine stores chunks and metadata locally.

Trust Graph

Graph model for panels, jobs, programs, datasets, and relationships with live updates.

TN3270 Stack

Wrapped TN3270 emulator for screen capture, key events, and session state management.

Recon Engine

Native Python enumeration of TSO users, CICS transactions, VTAM APPLIDs, hidden fields, and screen patterns. AI-assisted analysis via Ollama interprets findings and recommends next steps. Zero external dependencies — all I/O through the TN3270 stack.

Recon Engine Pipeline

The recon engine reimplements classic TN3270 enumeration scripts as native Python. All I/O flows through agent_tools.py — no subprocess calls, no external scripts.

Connect
TN3270 Session
Connect to target host:port via py3270 emulator.
Navigate
Command Sequence
Send configurable commands to reach the target screen (TSO logon, CICS region, VTAM).
Probe
Enumerate
Send each candidate (userid, transaction, applid) and read the response screen.
Classify
Pattern Match
Match screen text against known response codes (IKJ, DFHAC, IST messages).
Analyze
AI Interpretation
Ollama reviews findings, assesses risk, and recommends next steps.
Report
Export
Generate HTML, JSON, or Markdown reports with severity breakdowns.
