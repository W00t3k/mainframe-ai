System Architecture

A local-first mainframe operations assistant with FastAPI at the center, Ollama for LLM + embeddings, a file-backed RAG store, and a TN3270 pipeline for live terminal control.

Web UI

Jinja templates with shared CSS, toast notifications, and a consistent IBM-style layout.

API Layer

FastAPI endpoints for chat, terminal I/O, RAG ingestion, scanner operations, and screencaps.

LLM + RAG

Ollama handles generation and embeddings; the RAG engine stores chunks and metadata locally.

TN3270 Stack

BIRP v2 wraps a TN3270 emulator for screen capture, key events, and session state.

Request Flow

01 UI
