Getting Started

BigIron Control-Plane Assessment

Operator guidance for live mainframe engagements. Not a training course.

Engagement Scope

  • Map execution trust: VTAM → TSO → JES → RACF
  • Trace identity binding through deferred execution
  • Locate writable libraries in trusted job paths
  • Demonstrate control-plane privilege paths

Immediate Actions

  • Connect to TK5 and establish a TN3270 session
  • Bind identity: LOGON → TSO → RFE
  • Enumerate datasets and library resolution order
  • Submit a job and trace the execution path

Rules of Engagement

  • Scope: TK5 lab environment only
  • No memory exploits, no external tools
  • Normal user actions only — no system modification
  • Document execution paths and trust relationships

Environment Notes

  • MVS 3.8j — no shell, no processes, no fork
  • RFE (not ISPF), QUEUE (not SDSF)
  • PF3 to exit, PF7/PF8 to page
  • Never disconnect while logged in — locks the userid

Quick Start

  • LOGON USET → password PASS — bind identity
  • RFE option 3.4 — enumerate datasets
  • LISTCAT — catalog namespace inspection
  • LISTALC STATUS — current resource footprint
  • SUBMIT — deferred execution via JES
