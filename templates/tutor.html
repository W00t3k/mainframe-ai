Welcome to Red Team Tutor

Learn mainframe security through guided exploration.

Red Team Focus

  • Map the session stack (VTAM → TSO → ISPF)
  • Spot access controls (RACF, dataset profiles)
  • Find weak links (permissive datasets, defaults)
  • Capture evidence (screens, JES output, logs)

Starter Objectives

  • Connect to TK5 and reach the TSO logon
  • List jobs in JES spool and read output
  • Inspect dataset security metadata
  • Ask the tutor to explain a screen or error

Engagement Checklist

  • Scope: TK5 lab only, no external targets
  • Evidence: note steps, panels, timestamps
  • Minimize: change nothing unless instructed
  • Exit: document findings and clean up

Common Pitfalls

  • PF keys vary; try PF3 to exit, PF7/PF8 to page
  • Uppercase matters in TSO and dataset names
  • JES output appears after job completes
  • ISPF panels may hide commands in line 1

First 5 Commands

  • TSO: `LOGON USET` then password `PASS`
  • ISPF: `3.4` to browse datasets
  • TSO: `LISTCAT` to enumerate catalogs
  • JES: `SDSF` then `ST` for job status
  • ISPF: `6` for command shell
